**Starting the Server:**
1. Navigate to the server directory:
cd mern/server
2. Install the required dependencies:
npm install
3. Start the server:
npm start

**Starting the Web Client:**
1. Navigate to the client directory:
cd mern/client
2. Install the required dependencies:
npm install
3. Start the development server:
npm run dev
